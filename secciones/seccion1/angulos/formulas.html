Calcula el Ángulo

Datos del ejercicio:
Ángulo agudo: 42° Longitud del lado adyacente: 10 unidades

El ángulo que forman dos lados opuestos de un triángulo rectángulo es igual a la diferencia entre los grados del ángulo interior y el ángulo exterior

La relación trigonométrica del seno en un triángulo rectángulo es:
sin(θ)= Lado opuesto/ Hipotenusa

Código JavaScript para calcular el seno de un ángulo

                    function calculaSeno(angulo) {
                        return Math.sin(Math.PI * angulo / 180);
                    }
